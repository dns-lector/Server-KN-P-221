## Розробка серверних додатків
### Інструкція з клонування у частині Django
- Копіювати код (клонувати проєкт)
- Оновити (створити) віртуальне оточення 
    * Відкрити директорію, вищу за проєктну (django)
    * `python -m venv django`
    * Пересвідчитись у появі додаткових директорій
- Активувати віртуальне оточення
    * Переходимо до директорії проєкту `cd django`
    * `./Scripts/Activate.ps1`
    * Переконуємось у зміні промпта
- Відновлюємо залежності проєкта (пакети): `pip install Django`, 
переконуємось у появі директорій у `Lib`
- Переходимо до Django-проєкту (project221) `cd project221`
- Виконуємо міграції `python manage.py migrate`
- Запускаємо сервер, переходимо на браузер
`python manage.py runserver`
